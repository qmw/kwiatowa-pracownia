# Kwiatowa Pracownia

Przykładowa strona wizytówkowa dla fikcyjnej kwiaciarni **Kwiatowa Pracownia** — florystyka autorska z Krakowa. Projekt zbudowany jako pozycja portfolio.

## Styl

- **Paleta:** złamana biel #faf6f3, przygaszony róż #cf9b97, szałwiowa zieleń #9aaa8b, mauve #8a6d78, ciemny #3a2e31
- **Fonty:** Italiana (display, elegancki) + Jost (body, nowoczesny)
- **Charakter:** romantyczny, delikatny, kwiatowy — reveal przy scrollu, mikrointerakcje hover, galeria z filtrami

## Sekcje

1. Nawigacja fixed z CTA
2. Hero — pełnoekranowe zdjęcie z gradientem i overlayem
3. O Pracowni — historia, statystyki
4. Oferta — 5 usług z cenami (bukiety, śluby, wiązanki, kwiaty na okazje, abonament)
5. Opinie klientek
6. Galeria realizacji z filtrami (bukiety / śluby / kompozycje)
7. Formularz zamówienia
8. Kontakt + godziny + mapa OSM
9. Footer z creditem stronainternetowa.biz

## Stack

- [Astro 5](https://astro.build/) — framework
- [Tailwind CSS 4](https://tailwindcss.com/) — przez `@tailwindcss/vite`

## Komendy

```bash
npm install
npm run dev      # serwer lokalny
npm run build    # produkcyjny build do ./dist
npm run preview  # podgląd zbudowanej wersji
```

## Hosting

Projekt gotowy do wdrożenia na [Vercel](https://vercel.com/) — wystarczy połączyć repozytorium.
